221202 [6] Method

1. 메소드의 등장
객체지향 이전의 프로그래밍은 순차적으로 실행되는 절차지향 방식은 비효율적이며, 동일한 로직의 코드가 많아지고 코드의 양이 길어지기 때문에 유지보수가 어려움
=> 유지보수 좋고, 가독성 높은 메소드 등장

메소드란? 작업을 수행하기 위한 명령문의 집합으로, 어떤 값을 입력 받아서 처리하고 결과를 돌려줌 (입력 받는 값이 없을 수도 있고, 결과를 돌려주지 않을 수도 있음)

2. 메소드 작성 지침:
접근제한자 [static] 리턴타입 메소드명([매개변수1, 매개변수2 ...]){
	처리할 프로세스들
	return 리턴값;
}
> 리턴이 없을 경우 리턴타입은 void
