(2) switch문
switch(정수형 또는 문자형 변수 cf. 자바 1.7버전 이상은 문자열도 가능은 해){
	case 변수값1 : 위의 변수가 변수값1일 때 처리할 명령문; break;
	case 변수값2 : 위의 변수가 변수값1일 때 처리할 명령문; break;
	∙ ∙ ∙ ∙ ∙ ∙
	default : 위의 변수값들이 아닐 때 처리할 명령문; break;

* int temp = 임시변수
