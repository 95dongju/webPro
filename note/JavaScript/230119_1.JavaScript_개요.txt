230119 [1] JavaScript 개요

1. 자바 스트립트의 역사
- 넷스케이프 사에서 '모카'라는 이름으로 시작되었으며, 선 마이크로시스템에서 개발한 '자바' 프로그래밍 언어와 이름을 갖게 하는 마케팅 조약을 체결하여 JavaScript가 되었음


2. 자바 스크립트의 활용
- 웹 문서의 내용을 동적으로 바꾸거나, 사용자가 마우스를 클릭하는 등의 이벤트를 처리할 수 있게 함.
- 인터넷 속도가 빨라지며 최근의 웹 문서는 많은 자료를 포함하기 때문에, 웹 문서의 개념을 초월해 '웹 애플리케이션'이라 불리게 됨.
- 웹 애플리케이션은 노트북, 데스크탑, 스마트폰 등 모든 장치에서 사용 가능


3. HTML 파일 만들기
- <Script> 태그는 일반적으로 head 안에 위치한다


4. 오류 확인
- 크롬에서 F12를 눌러 경고를 확인할 수 있다

	elements : 페이지 내의 문서 객체와 관련된 모든 속성을 실시간으로 확인 가능
	resources : 페이지를 실행하면서 불러들인 모든 리소스 파일을 살펴볼 수 있음
	network : Ajax를 포함한 모든 http 요청을 확인 가능
	sources: 자바스크립트와 관련된 디버깅을 수행
	timeline: 각 요소에 어느 정도의 시간이 걸리는지 확인 가능
	profiles: 자바스크림트를 분석할 때 도움을 주는 기능
	audits: 페이지를 최적화하기 위한 정보 제공
	console: 콘솔 로그 확인 가능하며, 프로그램 실행된 상태에서 자바스크립트 코드 입력 가능
